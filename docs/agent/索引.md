## 设计原则

基于 HumanLayer 的 CLAUDE.md 最佳实践：

1. **Less is More**：核心 CLAUDE.md 控制在 60-80 行
2. **渐进式披露**：详细内容按需加载，避免上下文膨胀
3. **指针优于副本**：引用源代码路径而非复制代码片段
